
# Cipher Schools

The web app helps you learn through lectures posted by industry experts

## Prerequisites
- Nodejs Installed

- Npm Installed

- Suitable Text Editor

- Preferred Versions

node

```bash
  v18.14.0
```

npm

```bash
  v9.3.1
```



## Run Locally

Clone the project

```bash
  git clone git@github.com:kirti281003/SideMoney.git
```

Go to the project directory

```bash
  cd SideMoney
```

Install dependencies

```bash
  npm install
```

To Start The Frontend

- Go to client folder
```bash
  cd client
```
- Start the server on localhost:3000
```bash
  npm start
```

To Start The Backend

- Go to server folder in new terminal
```bash
  cd server
```
- Start the server on localhost:4000
```bash
  npm run dev
```
